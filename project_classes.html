Classes

  • AlgControlThread: controls retrieving frames of animation: stop, forward, reverse, step forward, step reverse, and the delay between frames of continuous animation.
  • AlgFrame: holds data for one frame of animation.
  • Algorithm: a generic algorithm.
  • AlgPanel: allows user choice of algorithm to simulate.
  • AlgSource: maintains sourcecode for the algorithm.
  • AlgTape: stores ordered frames of animation.
  • ButtonPanel: holds buttons that control algorithm features.
  • CodePanel: creates a canvas for drawing sourcecode text on.
  • Controller: vcr interface for algorithm playback.
  • DisplayCanvas: a double buffered canvas.
  • DisplayCanvasWireFrame: uses a 4 parameter viewing system to display wireframe images on a canvas.
  • DisplayFrame: holds DisplayFrameElements for one frame of a recording.
  • DisplayFrameElement: generic element to be drawn on a canvas.
  • Raytracer: an implementation of a raytracer.
  • RaytracerButtonPanel: extends ButtonPanel, changes the execute button to say "render".
  • RaytracerDFEBox: a wireframe box.
  • RaytracerDFELight: a wireframe light.
  • RaytracerDFELine: a wireframe line.
  • RaytracerDFESphere: a wireframe sphere.
  • RaytracerDFETriangle: a wireframe triangle.
  • RaytracerDFEView: wireframe viewing information.
  • RaytracerDisplayCanvas1: adds mouse clicking to the left canvas. used to specify the point to animate a rendering.
  • RaytracerFrame: a raytracer animation frame.
  • RaytracerSetupButtons: realestate on the destop for placing setup control buttons.
  • RaytracerSource: source code for a raytracer.
  • RaytracerState: overrides component choices for sim, help, setup, and pref panels.
  • SetupBoxPanel: controls modifying a box.
  • SetupButtonPanel: realestate on the destop for placing object setup buttons.
  • SetupColorPanel: controls modifying a color.
  • SetupElementsPanel: realestate on the destop for placing object setup panels.
  • SetupLightPanel: controls modifying a light.
  • SetupSpherePanel: controls modifying a sphere.
  • SetupTrianglePanel: controls modifying a triangle.
  • SetupViewPanel: controls modifying a view.
  • Sim: main class for simulation.
  • SimState: loads default messages and components for an algorithm.
  • T2d: holds an (x,y).
  • T3d: holds an (x,y,z).
  • TColor: holds an RGB color
  • TLight: holds an (x,y,z) and a brightness.
  • TMatrix3d: hold a 3x4 matrix
  • Utils: my un-sorted utilities.
