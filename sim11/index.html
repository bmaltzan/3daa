Getting Started:
  1. Press sim to go to the sim panel.
    Press render to execute the raytracer.
    An image will be created then displayed.
  2. Click on a pixel in the created image to trace.
    Press the record button.
    Use the control panel to move forward/backward through the recording.
  3. Press setup to switch to the setup panel.
    From objects, select new sphere.
    Move the radius bar up to 50 or so.
    Select a dark gray ambient color.
    Select a diffuse color.
    Select a specular color.
    Change the 'coef' value to 8 or so.
    Under lighting, select 'define new'.
    Change the y light point value to 200 or greater.
    Press save.
    Press sim.
    Press render.
