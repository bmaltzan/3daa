Setup Hints

Java 1.1

This version requires Netscape 4.07 or greater, or Internet Explorer 4.0 or greater. To determine the current version, go to the help->about box from your browser. To download a current version, follow one of these links: Netscape, IE

Java 1.2

This version require Netscape 4.5 or greater, or Internet Explorer 4.0 or greater, Java JRE 1.2(includes Java plug-in), and Java3d JRE 1.1 (either OpenGL or DirectX version).

Example installation:

  • download Netscape 4.5, Java JRE 1.2&Java plug-in, and Java3d JRE1.1
  • install netscape in the default directory.
  • install Java JRE 1.2 in the default directory.
  • install Java plug-in in the default directory.
  • install Java3d JRE in the Java JRE 1.2 directory.
  • copy j3daudio.jar, j3dcore.jar, j3dutils.jar, and vecmath.jar from
    c:\Program Files\JavaSoft\Jre\1.2\lib\ext to
    c:\Program Files\JavaSoft\Jre\1.2\lib
  • test your install. (~2K)
